### ✅ Задача 1: Статистика загрузок
**Статус:** Реализовано полностью

Добавлен класс `DownloadStats` для отслеживания:
- ⏱️ Время начала и окончания загрузки
- 📦 Общий объём скачанных данных (в GB/MB)
- 🚀 Средняя скорость загрузки (MB/s)
- 📈 Процент успеха (загружено/всего)
- ⏳ Оставшееся время до завершения (ETA)

**Методы класса:**
```python
.start() → засекает время начала
.finish() → засекает время окончания
.add_bytes(size) → добавляет загруженные байты
.get_duration() → возвращает длительность
.get_speed_mbps() → вычисляет среднюю скорость
.get_success_rate() → вычисляет % успеха
.get_eta(current, total) → вычисляет ETA
```

### ✅ Задача 2: Проверка свободного места на диске
**Статус:** Реализовано полностью

Функция `check_disk_space(target_dir)`:
- ✅ Показывает текущее свободное место (GB/MB)
- ⚠️ Выводит **ВНИМАНИЕ**, если < 500 MB
- ❌ **Останавливает загрузку**, если < 100 MB
- Вызывается автоматически перед каждой загрузкой

**Пример вывода:**
```
💾 Свободное место на диске: 45.23 GB (46320 MB)
⚠️  ВНИМАНИЕ: < 500 MB свободно! Рекомендуется освободить место.
❌ КРИТИЧНО: < 100 MB свободно! Загрузка остановлена!
```

### ✅ Задача 3: История загрузок (download_history.json)
**Статус:** Реализовано полностью

Создан файл `download_history.json` в базовой папке, содержащий:
- ✅ Какие URL уже обработаны
- ✅ Когда они были загружены (ISO формат)
- ✅ Количество файлов в каждой задаче
- ✅ Размер каждого файла
- ✅ Статус загрузки (наличие записи = успех)
- ✅ MD5 хеш каждого файла
- ✅ Исходный URL файла

**Функции:**
```python
load_download_history(base_dir) → загружает из JSON
save_download_history(base_dir, history) → сохраняет в JSON
```

**Формат JSON:**
```json
{
  "Task_Name_01": {
    "task_name": "Task_Name_01",
    "first_download": "2026-01-27T14:30:45.123456",
    "files": [
      {
        "filename": "Task_Name_01_12345.jpg",
        "size": 1048576,
        "md5": "5d41402abc4b2a76b9719d911017c592",
        "download_date": "2026-01-27T14:30:50.654321",
        "url": "https://example.com/download/12345"
      }
    ]
  }
}
```

### ✅ Задача 4: Пропуск дубликатов по хешу MD5
**Статус:** Реализовано полностью

Функции для проверки дубликатов:
```python
get_file_md5(file_path) → вычисляет MD5 хеш
check_duplicate_by_md5(file_path, base_dir, history) → проверяет наличие
```

**Работает следующим образом:**
1. При загрузке файла вычисляется его MD5 хеш
2. Хеш сравнивается со всеми файлами в истории
3. Если дубликат найден → логируется информация
4. Дубликат сохраняется в историю (для отслеживания)

**Преимущества:**
- 💾 Экономия трафика (дубликаты пропускаются)
- 🔒 Проверка целостности файлов
- 📋 Полная история загрузок
