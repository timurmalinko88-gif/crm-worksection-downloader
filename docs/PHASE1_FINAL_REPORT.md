## 🎨 ЦВЕТОВАЯ СХЕМА

Программа использует следующие цвета:

```
🟢 Зеленый (SUCCESS)      - ✅ успешные операции
🔴 Красный (ERROR)        - ❌ ошибки и проблемы  
🟡 Желтый (WARNING)       - ⚠️  предупреждения
🔵 Синий (INFO)           - ℹ️  информационные сообщения
🟦 Голубой (HEADER)       - 🎯 заголовки разделов
🟪 Пурпурный (MENU)       - 📋 элементы меню
```

### Примеры в коде
```python
from menu import Colors

# Успех
print(f"{Colors.SUCCESS}✅ Файл загружен{Colors.RESET}")

# Ошибка
print(f"{Colors.ERROR}❌ Ошибка загрузки{Colors.RESET}")

# Предупреждение
print(f"{Colors.WARNING}⚠️  Диск переполнен{Colors.RESET}")

# Информация
print(f"{Colors.INFO}ℹ️  Совет: используйте help{Colors.RESET}")
```

## ⌨️ ГОРЯЧИЕ КЛАВИШИ

| Клавиша | Место | Действие |
|---------|-------|----------|
| `1-7` | Главное меню | Выбор пункта |
| `back` | Любой диалог | Возврат в меню |
| `exit` / `quit` / `q` | Главное меню | Выход из программы |
| `help` / `h` / `?` | Выбор пункта | Справка по командам |
| `Ctrl+C` | Любое место | Прерывание + вопрос выхода |

### Примеры
```
> Выберите пункт (1-7): help
Команды:
  1-7    - Выберите пункт меню
  back   - Вернуться в меню
  exit   - Выход

> Выберите пункт (1-7): 1

> Введите URL: back
> Перешли в главное меню
```

## 🔧 ТЕХНИЧЕСКИЕ ДЕТАЛИ

### Новые зависимости
```python
colorama>=0.4.6  # Для цветного вывода
```

### Graceful Fallback
Если colorama не установлена, программа продолжит работать в обычном текстовом режиме без цветов.

```python
try:
    from colorama import Fore, Back, Style, init
    init(autoreset=True)
    COLORS_AVAILABLE = True
except ImportError:
    COLORS_AVAILABLE = False
    # Используются пустые строки вместо цветов
```

### Безопасность
- ✅ Все пользовательские входы валидируются
- ✅ Все файловые операции обёрнуты в try/except
- ✅ Нет hardcoded путей (используются переменные конфига)
- ✅ Все исключения логируются перед выводом
