## 📊 ЧТО ПРОИСХОДИТ ПРИ ЗАПУСКЕ

```
1. Проверяется свободное место на диске
   ├─ Если < 100 MB → загрузка ОСТАНАВЛИВАЕТСЯ
   └─ Если < 500 MB → выводится ПРЕДУПРЕЖДЕНИЕ

2. Загружается история из download_history.json

3. Инициализируется статистика (start_time, счётчики)

4. Для каждого файла:
   ├─ Скачивается файл
   ├─ Вычисляется MD5 хеш
   ├─ Обновляется статистика (добавляются байты)
   └─ Сохраняется в историю

5. По завершении выводится красивая статистика:
   ├─ Скачано файлов: X/Y
   ├─ Процент успеха: Z%
   ├─ Общий объём: A.B GB
   ├─ Время загрузки: C сек
   └─ Средняя скорость: D MB/s
```
